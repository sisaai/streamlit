## 🚀 실행 방법

### 1 Streamlit 실행  
아래 명령어를 실행하여 **Streamlit UI**를 실행하세요.

```bash
streamlit run deepseek_r1_streamlit_v1.py
```

### 2 Ollama 실행  
아래 명령어를 실행하여 **sisaai-llama3.1**를 실행하세요.

```bash
ollama run sisaai/sisaai-llama3.1
```
